# Platform Overview

PanEval is a dedicated large language model evaluation platform deployed by the Beijing Academy of Artificial Intelligence \(BAAI\) for the Eclipse Foundation in Europe\. As a bridge between advanced AI models and standardized evaluation systems, PanEval introduces an innovative “Capability–Task–Metric” evaluation framework\. Its goal is to establish scientific, objective, and efficient benchmarks and tooling to help researchers and partner developers comprehensively evaluate the performance of foundation models and training algorithms\.

In the first phase of PanEval, the system is primarily focused on providing automated and standardized online evaluation capabilities\. It currently supports two core areas: **Natural Language Processing \(NLP\) evaluation** and **Multimodal model evaluation**\. At this stage, the platform is designed exclusively for **API\-based online evaluation** and does not support local deployment or offline evaluation\.
