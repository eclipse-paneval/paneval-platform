# PanEval Phase I Evaluation Dataset List

# Dataset Overview

PanEval Phase I includes a total of **13 NLP model evaluation datasets **and **21 Multimodal model evaluation datasets**, with the specific list as follows:

## NLP Model Evaluation Datasets:

|**Dataset**|**Evaluation Framework**|**Dataset Description**|**Dataset Address**|
|---|---|---|---|

|BBH|Imeval|Reasoning ability – collection of challenging reasoning tasks for large models|[https://github\.com/suzgunmirac/BIG\-Bench\-Hard](https://github.com/suzgunmirac/BIG-Bench-Hard)|
|TheoremQA|Imeval|Mathematical ability – theorem\-based mathematical reasoning and QA|[https://github\.com/TIGER\-AI\-Lab/TheoremQA](https://github.com/TIGER-AI-Lab/TheoremQA)|
|GPQA|Imeval|Mathematical ability – graduate\-level difficult scientific questions|[https://github\.com/idavidrein/gpqa](https://github.com/idavidrein/gpqa)|
|LiveBench|Imeval|General ability – dynamically updated comprehensive evaluation|[https://github\.com/livebench/livebench](https://github.com/livebench/livebench)|
|MMLU\_Pro|Imeval|General ability – multidisciplinary professional knowledge QA|[https://github\.com/TIGER\-AI\-Lab/MMLU\-Pro](https://github.com/TIGER-AI-Lab/MMLU-Pro)|
